	ANSIREAD USAGE:

                                                                                                  
        usage: ansiread [options] <input-filename>
        
            (use "-" character for <input-filename> to read from stdin)
        
            -c              output in CP437 rather than UTF-8 (default)
            -f <output>     write output to file instead of stdout
            -g              enable (OpenGL) preview mode
            -o <output>     write PNG output to <output-png-filename> (not supported on Windows)
            -p              enable line padding (to 80 characters)
            -r              set enable-clear-mode (allow ANSI control codes that clear the canvas)
            -w              set enable-auto-line-wrap (for input that assumes tty is 80 columns)
            -z              enable compressed color codes for stdout or file output
            -d              enable debug mode
        
        

	TDFTOOLS USAGE:

        
        tdftool version 0.3.1
        
                                                                           
        usage: tdftool [options] <tdf-font> "your text here"
        
            -l              list mode
            -d              increase debugging level (use up to four times)
            -|              render vertically instead of horizontally
            -f <n>          use specified subfont
            -c              render to CP437 glyphs instead of UTF-8
            -s              append SAUCE record
            -v              show version information
